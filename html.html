About HTML

HTML stands for hypertext markup language and is the standard language for creating web pages which describes the structure of a webpage and consists of series of elements that tell the browser how to Display its contents which is why it should be used in every website. Examples of websites that used html include jumia, wish, w3schools, github and so on.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. By adding semantic tags to your document, you provide additional information about that document, which aids in communication. Specifically, semantic tags make it clear to the browser what the meaning of a page and its content is.
